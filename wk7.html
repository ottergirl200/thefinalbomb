Physical computing involves creating tangible, embedded microcontroller-based systems that interact with the physical world. Essentially, it is a discipline that combines hardware and software.

This may include the use of sensors and actuators that detect and control outputs such as lights, audio, displays etc.

First was a button that when pressed, the green bulb would light up.

Second was using the potentiometer to control the strength of the light.
